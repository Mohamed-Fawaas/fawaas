⚙️ How to Run
Option 1 — Direct Browser (Simple)
Extract the ZIP file
Open `index.html` in any modern browser (Chrome, Firefox, Edge)
Done ✅
Option 2 — XAMPP Local Server (Recommended for PDF viewing)
Copy the `final_portfolio/` folder into `C:/xampp/htdocs/`
Start Apache in XAMPP Control Panel
Open browser → `http://localhost/final_portfolio/`
All PDFs and images will load correctly ✅
Option 3 — VS Code Live Server
Open the folder in VS Code
Right-click `index.html` → Open with Live Server
Browser auto-opens at `http://127.0.0.1:5500/` ✅
